# Security Champions — Interactive Course

A hands-on, interactive training program for developers stepping into the Security Champion role.

## Course Structure

| Module | Topic | Parts |
|--------|-------|-------|
| **1** | Making the Security Champion | 1 |
| **2** | Secure SDLC | 1 |
| **3** | OWASP Top 10 | 5 lenses |
| **4** | Threat Modeling | 2 (Concepts + Case Study) |
| **5** | Supply Chain Security | 2 (Attack Surfaces + Hardening & SLSA) |
| **6** | Secret Management | 2 (Fundamentals + Cloud Solutions) |
| **7** | Security Configuration & Access Control | 2 (Configuration + Access Control) |
| **8** | Incident Response | 2 (Fundamentals + Operationalizing IR) |
| **9** | Mobile Application Security | 2 (Fundamentals + MASVS & Pitfalls) |

## How to Use

Each module is a standalone HTML file. Open it in any modern browser — no server required.

## Repository Layout

```
module-1-security-champion/       → Role, mindset, responsibilities
module-2-secure-sdlc/             → Integrating security into the dev lifecycle
module-3-owasp-top-10/            → 5 analytical lenses on the OWASP Top 10
module-4-threat-modeling/          → STRIDE, attack trees, hands-on case study
module-5-supply-chain/             → SCA, SBOM, SLSA framework
module-6-secret-management/        → Vaults, rotation, cloud-native solutions
module-7-security-config/          → Hardening, RBAC, policy-as-code
module-8-incident-response/        → IR lifecycle, playbooks, operationalization
module-9-mobile-security/          → OWASP MASVS, common mobile pitfalls
```
